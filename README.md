
Bufanyun Extend
================

这个是PHP一个扩展包，用于在创建新的跨框架项目时快速部署常用方法，实现项目快速落地。

Example
-------

```php
// 数组类
use bufanyun\extend\utils\ArrayUtil;
```


Installation
------------

Use [Composer] to install the package:

```bash
composer require bufanyun/extend:dev-master
```
